#### 智能计算器简介：

本实验设计的智能计算器是一个多功能计算工具，集成了四种常用的计算功能：
1. **简单四则运算计算器**：支持加、减、乘、除四种基本运算，能够处理小数运算，并包含除零错误处理。
2. **BMI计算器**：根据用户的体重和身高计算身体质量指数（BMI），并给出相应的健康建议。
3. **成绩统计**：输入多个学生的成绩，自动计算平均分、最高分、最低分，并根据平均分给出等级评定。
4. **简易房贷/存款利息计算器**：支持等额本息房贷计算和存款利息计算，帮助用户进行财务规划。

**数据类型分析：**
- 使用 `float` 类型存储数值（支持小数运算）
- 使用 `list` 类型存储成绩列表
- 使用 `str` 类型存储用户输入和运算符

**处理方法：**
- 使用函数封装每个功能模块，提高代码的可读性和可维护性
- 使用 `try-except` 结构进行异常处理
- 使用 `if-elif-else` 结构进行条件判断
- 使用 `for` 循环处理批量数据输入

**异常分析：**
- **除零错误**：在四则运算中，当除数为零时，程序会检测并提示用户
- **非数字输入**：在所有数值输入环节，使用 `try-except` 捕获 `ValueError` 异常
- **负数输入**：在房贷/存款计算中，验证输入值必须大于零
- **空列表错误**：在成绩统计中，验证学生人数必须大于零

import math


def simple_calculator():
    """
    简单四则运算计算器
    功能：加、减、乘、除
    数据类型：float（支持小数运算）
    异常处理：除零错误、非数字输入
    """
    print("\n=== 简单四则运算计算器 ===")

    try:
        # 获取用户输入的第一个数字
        num1 = float(input("请输入第一个数字: "))

        # 获取运算符
        operator = input("请输入运算符 (+, -, *, /): ")

        # 获取用户输入的第二个数字
        num2 = float(input("请输入第二个数字: "))

        # 根据运算符执行相应的运算
        if operator == '+':
            result = num1 + num2
            print(f"结果: {num1} + {num2} = {result}")
        elif operator == '-':
            result = num1 - num2
            print(f"结果: {num1} - {num2} = {result}")
        elif operator == '*':
            result = num1 * num2
            print(f"结果: {num1} * {num2} = {result}")
        elif operator == '/':
            # 处理除零错误
            if num2 == 0:
                print("错误：除数不能为零！")
            else:
                result = num1 / num2
                print(f"结果: {num1} / {num2} = {result}")
        else:
            print("错误：不支持的运算符！")

    except ValueError:
        print("错误：请输入有效的数字！")
    except Exception as e:
        print(f"发生未知错误: {e}")


def bmi_calculator():
    """
    BMI计算器（身体健康指数）
    功能：计算BMI值并给出健康建议
    数据类型：float（体重和身高）
    公式：BMI = 体重(kg) / 身高^2(m)
    异常处理：非数字输入、身高为零
    """
    print("\n=== BMI计算器 ===")

    try:
        # 获取用户输入的体重（kg）
        weight = float(input("请输入您的体重(kg): "))

        # 获取用户输入的身高（m）
        height = float(input("请输入您的身高(m): "))

        # 处理身高为零的错误
        if height == 0:
            print("错误：身高不能为零！")
            return

        # 计算BMI值
        bmi = weight / (height ** 2)

        # 输出BMI值
        print(f"\n您的BMI值为: {bmi:.2f}")

        # 根据BMI值给出健康建议
        if bmi < 18.5:
            print("健康状态: 偏瘦")
            print("建议: 适当增加营养摄入，加强锻炼")
        elif 18.5 <= bmi < 24:
            print("健康状态: 正常")
            print("建议: 保持良好的生活习惯")
        elif 24 <= bmi < 28:
            print("健康状态: 偏胖")
            print("建议: 控制饮食，增加运动量")
        else:
            print("健康状态: 肥胖")
            print("建议: 咨询医生，制定减肥计划")

    except ValueError:
        print("错误：请输入有效的数字！")
    except Exception as e:
        print(f"发生未知错误: {e}")


def grade_statistics():
    """
    成绩统计
    功能：计算平均分、等级、最高分、最低分
    数据类型：list（存储成绩）、float（计算结果）
    异常处理：非数字输入、空列表
    """
    print("\n=== 成绩统计 ===")

    try:
        # 获取用户输入的成绩数量
        count = int(input("请输入学生人数: "))

        if count <= 0:
            print("错误：学生人数必须大于0！")
            return

        # 创建空列表存储成绩
        grades = []

        # 循环获取每个学生的成绩
        for i in range(count):
            grade = float(input(f"请输入第{i+1}个学生的成绩: "))
            grades.append(grade)

        # 计算平均分
        average = sum(grades) / len(grades)

        # 计算最高分和最低分
        max_grade = max(grades)
        min_grade = min(grades)

        # 输出统计结果
        print("\n=== 成绩统计结果 ===")
        print(f"平均分: {average:.2f}")
        print(f"最高分: {max_grade}")
        print(f"最低分: {min_grade}")

        # 根据平均分给出等级
        if average >= 90:
            level = "优秀"
        elif average >= 80:
            level = "良好"
        elif average >= 70:
            level = "中等"
        elif average >= 60:
            level = "及格"
        else:
            level = "不及格"

        print(f"等级: {level}")

        # 输出所有成绩
        print(f"\n所有成绩: {grades}")

    except ValueError:
        print("错误：请输入有效的数字！")
    except Exception as e:
        print(f"发生未知错误: {e}")


def loan_interest_calculator():
    """
    简易房贷/存款利息计算器
    功能：计算等额本息还款或存款利息
    数据类型：float（金额、利率、时间）
    公式：
      - 等额本息月还款 = [本金×月利率×(1+月利率)^还款月数] ÷ [(1+月利率)^还款月数-1]
      - 存款利息 = 本金 × 年利率 × 存款年限
    异常处理：非数字输入、负数输入
    """
    print("\n=== 简易房贷/存款利息计算器 ===")

    try:
        # 让用户选择计算类型
        print("请选择计算类型:")
        print("1. 房贷计算（等额本息）")
        print("2. 存款利息计算")

        choice = input("请输入选择(1/2): ")

        if choice == '1':
            # 房贷计算
            print("\n=== 房贷计算 ===")

            # 获取贷款本金
            principal = float(input("请输入贷款本金(万元): "))

            # 获取年利率
            annual_rate = float(input("请输入年利率(%): "))

            # 获取贷款年限
            years = int(input("请输入贷款年限(年): "))

            # 验证输入
            if principal <= 0 or annual_rate <= 0 or years <= 0:
                print("错误：输入值必须大于0！")
                return

            # 转换单位
            principal = principal * 10000  # 转换为元
            monthly_rate = annual_rate / 100 / 12  # 月利率
            months = years * 12  # 还款月数

            # 计算等额本息月还款
            monthly_payment = (principal * monthly_rate * (1 + monthly_rate) ** months) / \
                             ((1 + monthly_rate) ** months - 1)

            # 计算总还款额和总利息
            total_payment = monthly_payment * months
            total_interest = total_payment - principal

            # 输出结果
            print("\n=== 房贷计算结果 ===")
            print(f"贷款本金: {principal/10000:.2f} 万元")
            print(f"年利率: {annual_rate}%")
            print(f"贷款年限: {years} 年")
            print(f"月还款额: {monthly_payment:.2f} 元")
            print(f"总还款额: {total_payment/10000:.2f} 万元")
            print(f"总利息: {total_interest/10000:.2f} 万元")

        elif choice == '2':
            # 存款利息计算
            print("\n=== 存款利息计算 ===")

            # 获取存款本金
            principal = float(input("请输入存款本金(元): "))

            # 获取年利率
            annual_rate = float(input("请输入年利率(%): "))

            # 获取存款年限
            years = int(input("请输入存款年限(年): "))

            # 验证输入
            if principal <= 0 or annual_rate <= 0 or years <= 0:
                print("错误：输入值必须大于0！")
                return

            # 计算存款利息
            interest = principal * (annual_rate / 100) * years

            # 计算本息合计
            total = principal + interest

            # 输出结果
            print("\n=== 存款利息计算结果 ===")
            print(f"存款本金: {principal:.2f} 元")
            print(f"年利率: {annual_rate}%")
            print(f"存款年限: {years} 年")
            print(f"存款利息: {interest:.2f} 元")
            print(f"本息合计: {total:.2f} 元")

        else:
            print("错误：无效的选择！")

    except ValueError:
        print("错误：请输入有效的数字！")
    except Exception as e:
        print(f"发生未知错误: {e}")


def main():
    """
    主函数：提供用户交互界面
    功能：显示菜单，根据用户选择调用相应的函数
    """
    print("=" * 50)
    print("       智能计算器 - Python课程实验报告1")
    print("=" * 50)

    while True:
        # 显示主菜单
        print("\n请选择要执行的操作:")
        print("1. 简单四则运算计算器")
        print("2. BMI计算器")
        print("3. 成绩统计")
        print("4. 简易房贷/存款利息计算器")
        print("0. 退出")

        # 获取用户选择
        choice = input("\n请输入您的选择(0-4): ")

        # 根据用户选择调用相应的函数
        if choice == '1':
            simple_calculator()
        elif choice == '2':
            bmi_calculator()
        elif choice == '3':
            grade_statistics()
        elif choice == '4':
            loan_interest_calculator()
        elif choice == '0':
            print("\n感谢使用智能计算器，再见！")
            break
        else:
            print("错误：无效的选择，请重新输入！")


if __name__ == "__main__":
    main()

## 问题及思考：

### 实验总结：

通过本次实验，我成功设计并实现了一个功能完善的智能计算器，包含了四项核心功能：简单四则运算、BMI计算、成绩统计和房贷/存款利息计算。在实验过程中，我深入理解了Python编程的多个重要概念，并积累了宝贵的实践经验。

#### 1. 数据类型和运算符的应用

在实验中，我使用了多种数据类型：
- **float类型**：用于存储数值，支持小数运算，确保计算精度
- **list类型**：用于存储成绩列表，方便批量处理数据
- **str类型**：用于存储用户输入和运算符

通过实际应用，我深刻理解了不同数据类型的适用场景和转换方法。例如，在四则运算中，使用`float()`函数将字符串转换为浮点数，确保能够处理小数运算。

#### 2. 函数的定义和调用

我将每个功能模块封装为独立的函数，如`simple_calculator()`、`bmi_calculator()`等。这种模块化的设计带来了以下好处：
- **代码可读性**：每个函数功能单一，逻辑清晰
- **代码可维护性**：修改某个功能时，只需修改对应的函数
- **代码复用性**：函数可以在主程序中多次调用

通过实践，我掌握了函数的定义方法、参数传递和返回值的使用。例如，`grade_statistics()`函数不需要返回值，直接在函数内部输出结果；而其他函数也可以根据需要返回计算结果。

#### 3. 异常处理的重要性

在实验中，我遇到了多种异常情况：
- **除零错误**：在四则运算中，当除数为零时，程序会崩溃
- **非数字输入**：用户输入字母或符号时，`float()`函数会抛出`ValueError`异常
- **负数输入**：在房贷/存款计算中，输入负数会导致计算结果错误

通过使用`try-except`结构，我成功捕获并处理了这些异常，使程序更加健壮。例如：

```python
try:
    num1 = float(input("请输入第一个数字: "))
except ValueError:
    print("错误：请输入有效的数字！")
```

这种异常处理机制不仅提高了程序的稳定性，还改善了用户体验，让用户知道错误的原因。

#### 4. 选择语句和循环语句的应用

在实验中，我大量使用了`if-elif-else`结构进行条件判断：
- 在四则运算中，根据运算符选择不同的计算方式
- 在BMI计算中，根据BMI值给出不同的健康建议
- 在成绩统计中，根据平均分给出不同的等级评定

同时，我使用了`for`循环处理批量数据：
- 在成绩统计中，循环获取每个学生的成绩
- 在主菜单中，使用`while True`循环实现持续交互

通过实践，我深刻理解了选择语句和循环语句的使用场景和注意事项。

#### 5. 变量的作用域

在实验中，我注意到了变量的作用域问题：
- **局部变量**：在函数内部定义的变量，只能在函数内部使用
- **全局变量**：在函数外部定义的变量，可以在整个程序中使用

例如，在`grade_statistics()`函数中，`grades`列表是局部变量，只能在函数内部访问。如果需要在函数之间共享数据，可以通过参数传递或返回值实现。

#### 6. 遇到的问题及解决方案

**问题1：除零错误**
- **现象**：在四则运算中，当除数为零时，程序崩溃
- **解决**：在除法运算前，检查除数是否为零，如果为零则提示用户

**问题2：非数字输入**

- **解决**：使用`try-except`结构捕获`ValueError`异常，提示用户输入有效数字

**问题3：房贷计算公式错误**

- **解决**：查阅资料，确认正确的公式，并仔细检查代码实现

**问题4：BMI计算结果精度问题**
- **现象**：BMI值显示过多小数位
- **解决**：使用`f"{bmi:.2f}"`格式化输出，保留两位小数

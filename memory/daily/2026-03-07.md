## 技术收获

1. **Vue 3 props 只读规则**
   - v-model 不能用于 prop
   - 应该使用 :model-value + @update:modelValue

2. **Dev-Team workflow 设计**
   - 统一使用 subagent
   - 持续输出进度（每30秒）
   - 明确完成标志（DEV COMPLETE）

3. **构建清理策略**
   - 保留 node_modules（快速构建）
   - 删除构建产物（节省空间）

4. **Java/Spring Boot 开发规范**
   - 禁止 JS 风格代码
   - 必须同步更新 pom.xml
   - DTO 字段必须完整
   - API 统一使用 /api 前缀
